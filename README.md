A growing collection of scripts to perform various cleaning, organization and maintenance tasks in my homelab and help me get better at writing Python.

CONTENTS: 

- organize_downloads:
  A simple script that sorts the contents of my downloads folder by file type to make it more manageable
remove_zips: A script to recursively delete all .zip files in the selected directory and all subdirectories.

- remove_zips:
  I download a lot of 3D models that come in .zip format and my 3D printing folder was littered with extraneous .zip files that I no longer need.  This script recursively traverses my 3D printing folder and deletes all of those old .zip files that I don't need anymore.
